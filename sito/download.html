El Merendero


Scarica ultima versione (0.2)


El Merendero è una applicazine java creata per gestire l'ordine di merende e bevande dal bar del propio istituto senza scomodarsi recandovisi personalmente. Esso si rivela estremamente utile in ossasione delle prove d'esame in cui non è permesso a nessun studente di lasciare la propria classe negandogli la possibilità di concedersi una pausa mentre si gustano la loro merenda preferita.

Istruzioni d'uso

1. Come avviare un file .jar : un file di estensione .jar è un eseguibile in java, per questo per poter avviare questo tipo di estensioni bisogna installare sul proprio computer il java runtime environment (JRE) ---------> download ------> http://www.oracle.com/technetwork/java/javase/downloads/index.html

2. Download: click sul bottone Download sotto la scritta JRE (non server JRE) e seguire le istruzioni per il completamento del download.

3. Aprire il file El_Merendero.jar : in windows basta fare doppio click sul file scelto. Altrimenti per avviarlo usare la seguente istruzione -----> "Java -jar nomefile.jar".
